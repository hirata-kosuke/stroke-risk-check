# 脳卒中リスクスコアの配分

このファイルは、JPHC研究に基づく正確なスコア配分を記載します。

## 出典

国立がん研究センター JPHC研究
- https://epi.ncc.go.jp/jphc/outcome/3284.html
- https://epi.ncc.go.jp/riskcheck/str/

## スコア計算の流れ

### ステップ1: 点数を当てはめる
各リスク因子の点数を以下の表から取得

### ステップ2: 点数を合計する
全ての因子の点数を合計

### ステップ3: 10年間で脳卒中を発症する確率(%)を算出
合計点数から発症確率テーブルを参照

## リスク因子ごとのスコア配分

### 1. 年齢スコア
```
40-44歳: 0点
45-49歳: 5点
50-54歳: 6点
55-59歳: 12点
60-64歳: 16点
65-69歳: 19点
```

### 2. 性別スコア
```
男性: 6点
女性: 0点
```

### 3. 喫煙スコア（たばこを吸っている）
```
男性の場合: 4点
女性の場合: 8点
吸わない/過去: 0点
```

### 4. BMIスコア（肥満度）
```
BMI < 25: 0点
25 ≦ BMI < 30: 2点
BMI ≧ 30: 3点
```
※ BMI = 体重(kg) ÷ 身長(m) ÷ 身長(m)

### 5. 糖尿病スコア
```
あり: 7点
なし: 0点
```
※ 糖尿病ありとは: 治療中またはHbA1c(空腹時血糖値126 mg/dL以上)

### 6. 血圧スコア

#### 降圧薬内服なしの場合
```
<120/80: 0点
120-129/80-84: 3点
130-139/85-89: 6点
140-159/90-99: 8点
160-179/100-109: 11点
180-/110-: 13点
```

#### 降圧薬内服中の場合
```
<120/80: 10点
120-129/80-84: 10点
130-139/85-89: 10点
140-159/90-99: 11点
160-179/100-109: 11点
180-/110-: 15点
```

## 検証例

### 例1: 50歳男性、喫煙、BMI27.3、糖尿病あり、降圧剤服用
- 年齢(50歳): 6点
- 性別(男性): 6点
- 喫煙(現在): 4点
- BMI(27.3): 2点
- 糖尿病: 7点
- 降圧剤服用: 10点
- **合計: 35点 → 10-12%** ✓

### 例2: 58歳男性、喫煙、BMI26、糖尿病あり、血圧145/89
- 年齢(58歳): 12点
- 性別(男性): 6点
- 喫煙(現在): 4点
- BMI(26): 2点
- 糖尿病: 7点
- 血圧(140-159/90-99、降圧剤なし): 8点
- **合計: 39点 → 12-15%** ✓
